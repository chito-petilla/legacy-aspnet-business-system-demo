# Legacy ASP.NET Business System Demo
Sample ASP.NET Web Forms business application demonstrating accounting and management system architecture.

This repository demonstrates the architecture and development approach used in maintaining and enhancing long-term business applications built using:

- ASP.NET Web Forms
- C#
- SQL Server
- ADO.NET
- Stored Procedures

## Functional Areas

- User Management
- Customer Management
- Financial Transactions
- Reporting
- Audit Logging

## Key Responsibilities

- Maintenance and support
- Database optimization
- Bug fixing
- Feature enhancements
- Production deployment

## Technologies

- ASP.NET Framework
- C#
- SQL Server
- JavaScript
- HTML/CSS

## Notes

Business logic and client-specific source code have been removed to protect confidentiality.
